# Markdown 语法

- **粗体** (bold)
> 用 \*\*content\*\* 或者 \_\_content\_\_ 实现，中间不能有空格

- *斜体* (italic)
> 用\*content\ 或者 \_content\_实现 ，中间不能有空格
- ***粗斜体***(BoldAndItalic) 
> 等于上面之和，即\*\*\*content\*\*\* 或 \_\_\_content\_\_\_
- ~~删除线~~
> 使用\~\~content\~\~ 即可实现
